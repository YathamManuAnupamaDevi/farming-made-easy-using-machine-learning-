# farming-made-easy-using-machine-learning-
Machine Learning project for crop price prediction using Python and ML algorithms.

## Description
This project is a Machine Learning-based web application that helps farmers by predicting crop prices based on input data like weather and market conditions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

## Features
- Crop price prediction using ML models
- Data preprocessing and feature selection
- User-friendly web interface for farmers and admins
- Multiple ML algorithms used for better accuracy

## Algorithms Used
- Decision Tree
- K-Nearest Neighbors (KNN)
- Random Forest

## Project Structure
- Backend: Python
- Frontend: HTML, CSS
- Database: MYSQL

## How to Run
1. Install required libraries
2. Run the Python file
3. Open the application in browser
